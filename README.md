# 🎮 GameDex

**Project Overview:** A comprehensive game cataloging and discovery platform engineered to provide users with a personalized experience for tracking gaming interests, managing wishlists, and exploring game metadata.  
**Tech Stack:** PHP, JavaScript (AJAX), MySQL, CSS, HTML

## 📌 Overview

GameDex is a web application designed to bridge the gap between casual gaming interest and organized cataloging. By utilizing a robust PHP based backend and a responsive front end, the platform allows users to browse a dynamic database of titles, filter games by platform or genre, and manage a personal wishlist. The project focuses on seamless asynchronous data handling, secure user session management, and intuitive information architecture.

## Core Implementation

* **Asynchronous Interaction:** Implemented dynamic filtering and searching via AJAX, allowing for real time updates of game grids without page reloads.
* **Backend Logic:** Utilized a clean, model driven architecture in PHP to manage complex game relationships, platform mapping, and user specific wishlist states.
* **Database Management:** Integrated MySQL with structured SQL queries to ensure high performance retrieval of game data, ratings, and relational metadata.

## Key Features

* **Dynamic Game Filtering:** Advanced search functionality that enables users to filter, sort, and discover games based on specific platforms and genres in real time.
* **Interactive Wishlist:** A user centric feature allowing authenticated users to toggle and track their favorite titles using an event driven AJAX architecture.
* **Metadata Rich Presentation:** Structured cards displaying essential game information, including cover art, platform availability, and aggregate user ratings, designed for optimized readability.
